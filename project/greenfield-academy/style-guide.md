# Greenfield Academy (v2) - Style Guide

## Layout

The design uses a max-width container:

- Container max-width: 1000px
- Container padding: 0 20px

## Colors

### Primary

- **Primary (Forest Green)**: #228B22
- **Accent (Golden Yellow)**: #FFB703

### Neutral

- **Dark Green**: #1B4332
- **Header/Navbar**: #0F261A
- **Text**: #333333
- **Secondary (Light Green)**: #E8F5E9
- **Background**: #FAFAFA
- **White**: #FFFFFF
- **Border Gray**: #DDDDDD

### Overlays

- **Hero Overlay**: rgba(0, 0, 0, 0.6)
- **Text Shadow**: rgba(0, 0, 0, 0.5)

## Typography

### Body Copy

- Font size: 16px
- Line height: 1.6

### Headings

- H1: 32px (36px in hero)
- H2: 24px
- H3: 20px

### Font

- Family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif
- Weights: 400 (normal), 700 (bold)

## Spacing

- Section margin-bottom: 40px
- Paragraph margin-bottom: 15px
- Container padding: 20px
- Header/Footer padding: 15px / 25px

## Components

### Hero (v2 Enhanced)

- Background image with 60% dark overlay
- Min-height: 350px
- Text shadow: 2px 2px 4px rgba(0, 0, 0, 0.5)
- CTA button included

### Page Header (Inner Pages)

- Background: #228B22
- Text: #FFFFFF

### Buttons

- Background: #228B22
- Text: #FFFFFF
- Padding: 12px 25px
- Border radius: 5px
- Hover background: #FFB703
- Hover text: #1B4332

### Cards/Features

- Background: #E8F5E9
- Padding: 20px
- Border-left: 4px solid #228B22
- Border radius: 8px

### Info Section

- Background: #E8F5E9
- Padding: 25px
- Border-left: 4px solid #228B22
- Border radius: 8px

### Forms

- Input padding: 12px
- Border: 2px solid #DDDDDD
- Border radius: 5px
- Focus border: #228B22
